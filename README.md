# discovery-4-4ini
Discovery service for 4ini :-)

Require:
* Python >= 3.4
* virtualenv

```bash
> git clone https://github.com/simongsr/discovery-4-4ini.git
> cd discovery-4-4ini
> virtualenv --python=python3 .
> source bin/activate
> pip install -r requirements.txt
> python example_service.py  # or main.py
```
